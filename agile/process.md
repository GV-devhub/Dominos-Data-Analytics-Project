# Agile Process – Domino’s Pizza Analytics Project

This project follows a lightweight Agile approach suitable for analytics and BI development. The goal is to deliver value incrementally while maintaining clarity and adaptability.

---

## 1. Project Approach

The work is organized into short, focused iterations. Each iteration delivers a meaningful piece of the analytics solution:

- Clean and prepare data
- Build SQL analysis
- Create KPI views
- Develop Excel dashboard
- Implement RBAC model
- Document business insights

---

## 2. Sprints Overview

### **Sprint 1 – Data Understanding & Preparation**
- Review dataset structure
- Define business questions
- Create data model documentation
- Build Oracle table schema
- Clean and standardize raw data

### **Sprint 2 – SQL Analysis & KPIs**
- Write exploratory SQL queries
- Build KPI views for dashboards
- Validate calculations and logic
- Prepare processed datasets for Excel

### **Sprint 3 – Dashboard & Insights**
- Build Excel dashboard (KPIs, charts, slicers)
- Document insights and findings
- Align visuals with business questions

### **Sprint 4 – RBAC & Security**
- Define roles and permissions
- Create RBAC model documentation
- Write SQL examples for role creation

### **Sprint 5 – Final Documentation & Polishing**
- Update README
- Organize folder structure
- Add Agile and BA documentation
- Final review for portfolio readiness

---

## 3. Agile Artifacts

- **User Stories:** Captured in `agile/user_stories.md`
- **Backlog:** GitHub Issues
- **Kanban Board:** GitHub Projects (Backlog → In Progress → Done)
- **Documentation:** Stored in `/docs`, `/agile`, `/rbac`

---

## 4. Definition of Done (DoD)

A task is considered complete when:

- Code runs without errors
- Documentation is updated
- Output is validated
- Commit message is clear
- Stakeholder need (user story) is satisfied

---

## 5. Benefits of Agile for This Project

- Clear alignment with business needs  
- Incremental delivery of insights  
- Easy to extend with new features  
- Transparent progress tracking  
- Strong portfolio storytelling  

---

This Agile process demonstrates structured thinking, planning, and delivery — key skills for Business Analyst and Data Analyst roles.
